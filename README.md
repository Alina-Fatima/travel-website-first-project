# 🌍 World Travel — Travel Website

A responsive, multi-page travel website concept designed to showcase destinations, travel packages, and easy visitor engagement through a clean, modern interface.

## 📖 Description

World Travel is a front-end travel website built to explore layout design, image-heavy UI, and creating an inviting user experience for a travel/tourism business. It includes a home page with featured destinations, an about page with company stats, and a contact page with a working form layout.

## ✨ Features

- **Responsive navigation bar** with active page highlighting
- **Hero header section** with call-to-action button
- **Destination showcase** with image and description cards
- **Features section** highlighting service USPs (Adventure, Pricing, Experience)
- **About page** with company stats (places visited, guides, sports activities)
- **Contact page** with a styled form (name, email, phone, message) and company info sidebar
- **Social media footer** with icon links
- Clean, consistent styling across all pages using a shared stylesheet

## 🛠️ Tech Stack

- **HTML5** — semantic page structure
- **CSS3** — custom styling, Flexbox layouts, gradients, hover effects
- **Font Awesome** — icons throughout the site

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/travel-website.git
   ```
2. Navigate into the project folder:
   ```bash
   cd travel-website
   ```
3. Open `index.html` directly in your browser, **or** run it with a live server:
   - If using VS Code, install the **Live Server** extension and click "Go Live"
   - Or use any local server of your choice

No build tools or dependencies required — this is a pure HTML/CSS project.
